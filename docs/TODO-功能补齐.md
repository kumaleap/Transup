## 1. 运行时韧性 ✅（2026-07-08 完成，详见 ROADMAP M4.5）

现代 harness 的真实杀手不是缺功能，而是 agent loop 在长任务里断档。

- [x] **防断档续跑**：max_tokens 截断自动续跑 + 空回复自动催跑（每轮上限 3 次）。意图级检测（"接下来我会…"却停手）风险高，推迟到有真实使用数据再做。

- [x] **循环保护**：调用+结果签名检测，3 次警告、5 次熔断（loop_detected）。

> 实现落点：engine.ts 拆分为 compact / tool-runner / guard 三模块 + 主循环编排，文件保持精简。
